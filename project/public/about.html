About This Project

This interactive web application explores South Africa's socioeconomic landscape using data from the General Household Survey 2023. It provides insights into living conditions, income distribution, access to services, and more, empowering users to engage with the data and understand the complexities of South African society.

Project Goals

  • Broad Representation: The application covers critical themes like food security, income, education, health, and demographics, ensuring diverse and user-customizable insights.
  • Scholarly Support: The visualisations are grounded in scholarly research, promoting civic engagement and policy impact by making complex data accessible and understandable (Rosling et al., 2018).
  • Dynamic Usage: The application leverages interactive elements like sliders and toggles to allow users to explore data dynamically and uncover meaningful patterns (Chen, 1976).

Target Audience

This application is designed for the general public, aiming to make complex socioeconomic data accessible and engaging for everyone.

Visual Style

The interface follows a clean and minimalist aesthetic, prioritizing the visualizations and ensuring a clear and uncluttered presentation. Colour choices are inspired by the South African flag, adding a subtle thematic touch.

Visualisations

The application incorporates a variety of interactive visualisations, including:

  • Interactive Maps (Choropleth): Display regional disparities in key indicators across South African provinces.
  • Bar Charts: Compare categorical data and distributions.
  • Line Charts: Track trends over time.
  • Bubble Charts: Visualise correlations between three variables.
  • Heatmaps: Analyse correlations and distributions for two categorical variables.

Interactive Features

The application offers a range of interactive features, such as:

  • Map Interaction: Zoom, pan, and hover over provinces to view detailed information.
  • Dynamic Filters: Filter data by province, geographic type, income, and other relevant variables.
  • Chart Interaction: Access tooltips on charts to reveal precise data values and additional details.
  • Data Comparison Tools: Compare data across different provinces or categories.
  • Guided Exploration Mode: A step-by-step walkthrough of key visualisations with explanations.

Data Source

The data used in this application is sourced from the General Household Survey 2023 conducted by Statistics South Africa. You can access the data and metadata at the following links:

  • Statistics South Africa
  • Nesstar Data Browser
